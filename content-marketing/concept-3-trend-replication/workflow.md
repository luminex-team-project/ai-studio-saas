# Concept 3: 틱톡/릴스/숏츠 트렌드 복제

## 목표

트렌드 발견부터 게시까지 **24시간 이내** 완료. 속도 = 알고리즘 노출 = 신규 팔로워 유입.

## 핵심 원칙

1. **속도 > 완성도** — 트렌드 시점이 곧 가치
2. **AI 메타 활용** — "AI가 따라한 트렌드"라는 각도가 차별화
3. **자체 SaaS 도그푸딩** — 트렌드 복제용 templates (dance-challenge, pose-transition, rotate-360, reaction-challenge) 적극 활용

## 단계별 SOP

### 단계 0. 트렌드 모니터링 (주 1회 화요일 1시간)

[../shared/music-trend-tracker.md](../shared/music-trend-tracker.md) 참조.

수요일 게시할 트렌드를 화요일 저녁까지 픽 → 주간 트렌드 보드에 기록.

### 단계 1. 트렌드 분해 (영상당 10분)

4요소 워크시트 채우기:

| 요소 | 체크 |
|------|------|
| 사운드 | 곡 + 사용 구간 (예: 0:15~0:25) + BPM |
| 동작 | 키 동작 3개 + 타이밍 |
| 컷 구조 | 컷 횟수 + 전환 방식 |
| 텍스트 오버레이 | 위치 + 등장 타이밍 |

### 단계 2. 복제 모드 결정 (영상당 5분)

**모드 A: 자체 SaaS 템플릿 활용 (가장 빠름, 추천)**
- ai-studio-saas의 templates 중 매칭되는 것 픽:
  - dance-challenge → 댄스 챌린지
  - pose-transition → 포즈 트랜지션
  - rotate-360 → 360 회전
  - reaction-challenge → 리액션
- 셀피 1장 + 템플릿 선택 → 10 크레딧으로 5초 영상 자동 생성
- 소요: 10분

**모드 B: 본인 + AI 합성 (개성 강조)**
- 본인이 일부 동작 직접 촬영 (5초)
- Kling i2v로 배경/의상만 AI로 변형
- 소요: 30~45분

**모드 C: 풀 AI 변환 ("AI가 따라한 트렌드" 메타)**
- Luma Ray 2 t2v로 컨셉 자체를 AI 미학으로 재해석
- 예: 댄스 트렌드 → 로봇이 추는 버전 / 픽사 스타일 / 사이버펑크 스타일
- 소요: 60~90분 (가장 와우 효과 큼)

### 단계 3. 영상 생성

#### 모드 A 워크플로우
1. 자체 SaaS 로그인 → /create/selfie 접속
2. 매칭 템플릿 픽 (templates 테이블 참조)
3. 본인 셀카 또는 AI 모델 캐릭터 이미지 업로드
4. example_prompt에 트렌드 특화 문구 추가 (예: "원본 안무 그대로, 표정은 자신감 있게")
5. 생성 → 1~3분 대기 → 다운로드

#### 모드 B 워크플로우
1. 본인이 동작 직접 촬영 (스마트폰 9:16, 5~10초)
2. CapCut에서 키 프레임 추출 (3장)
3. 각 프레임을 Midjourney img2img로 배경/의상 교체
4. 새 키프레임을 Kling 2.5 Pro i2v로 영상화
5. CapCut에서 원본 동작 + AI 변형 컷 교차

#### 모드 C 워크플로우
1. 트렌드 컨셉을 텍스트 1줄로 정리 (예: "POV 댄스 챌린지를 사이버펑크 로봇이 따라한다면")
2. Luma Ray 2 t2v 프롬프트 작성 (영어, 30~50 단어)
3. 5초 생성 × 2~3개 (다른 컷)
4. CapCut에서 비트 매칭

### 단계 4. 사운드 (필수)

**TikTok/Reels:** 앱 내에서 직접 사운드 픽 (mp3 임포트 ❌)
**YouTube Shorts:** YouTube 라이브러리 또는 트렌드 사운드 매칭

### 단계 5. 편집 — CapCut (영상당 10~15분)

**컷 구조 (모드 A 기준):**
```
[0~1초]  훅 자막 ("AI한테 이거 시켜봤음")
[1~6초]  자체 SaaS 생성 영상 (5초)
[6~10초] 비교/리액션 컷 (선택)
[10~15초] CTA 자막 ("프로필 링크에서 직접 만들어보세요")
```

**모드 C 와우 컷 구조:**
```
[0~1초]   원본 트렌드 1초 인서트 + "AI 버전 봐봐"
[1~6초]   AI 변환 영상
[6~10초]  원본 vs AI 분할 화면 비교
[10~12초] CTA
```

### 단계 6. 게시 (영상당 5분)

- 캡션: "AI한테 [트렌드명] 시켜봤더니..."
- 해시태그: hashtag-library.json의 `concept_3_trend` + 당일 트렌드 해시태그 (반드시 트렌드 해시태그 포함)
- 게시 시각: 수요일 21:00 KST
- **트렌드 사운드 링크에 자동 등록됨** → 알고리즘 부스트 자동

## 모드별 비용 비교

| 모드 | 비용 | 소요 시간 | 와우 효과 | 차별화 |
|------|------|----------|----------|--------|
| A (자체 SaaS) | 10 크레딧 (≈ 330원) | 20분 | ★★★ | ★★ |
| B (본인+AI) | 25 크레딧 (≈ 825원) | 45분 | ★★★★ | ★★★★ |
| C (풀 AI 변환) | 50 크레딧+Luma (≈ 2000원) | 90분 | ★★★★★ | ★★★★★ |

**추천 비율:** A 60% / B 30% / C 10% (월 4개 트렌드 영상 기준)

## 트렌드 vs 자체 SaaS 템플릿 매칭

| 트렌드 패턴 | 자체 SaaS 템플릿 (slug) |
|------------|------------------------|
| 댄스 챌린지 | dance-challenge |
| 변신/포즈 변환 | pose-transition |
| 360도 회전 트렌드 | rotate-360 |
| 놀람/리액션 | reaction-challenge |
| 일상 브이로그 | daily-vlog |
| 슬로우 워크 | slow-motion-walk |
| 패션 워크 | fashion-walk |
| 언박싱 | product-unboxing |
