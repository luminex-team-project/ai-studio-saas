# Concept 4: AI 핫이슈/뉴스 빠른 큐레이션

## 목표

AI 뉴스 발생 후 **12시간 이내** 60초 영상으로 큐레이션 게시. "한국에서 빠르게 이해하기 쉽게 정리해주는 AI 채널" 포지셔닝.

## 결과물 스펙

| 항목 | 값 |
|------|----|
| 길이 | 50~70초 (스위트스팟 60초) |
| 비율 | 9:16 |
| 컷 수 | HeyGen 얼굴 1컷 (작게) + B-roll 4~6컷 (메인) |
| 보이스 | HeyGen TTS 또는 본인 보이스 |
| BGM | -22dB 미니멀 비트 |

## 단계별 SOP

### 단계 0. 뉴스 모니터링 (매일 아침 30분)

[../shared/ai-news-sources.md](../shared/ai-news-sources.md) 참조.

X 알림 + Reddit 핫 + 한국 뉴스 1차 스캔 → 큐레이션 점수표 (5항목 합산 30점 이상)에 통과한 뉴스 선정.

### 단계 1. 스크립트 작성 — ChatGPT (영상당 10분)

ChatGPT 프롬프트 템플릿:

```
다음 AI 뉴스를 60초 한국어 숏폼 영상 스크립트로 변환해줘.

[뉴스 URL 또는 본문 붙여넣기]

요구사항:
- 총 150 단어 내외
- 구조: [0~5초 훅] [5~45초 본문 3포인트] [45~60초 CTA]
- 훅: 충격적/궁금한 한 줄 (예: "이거 진짜 미쳤음")
- 본문: ① 무엇이 달라졌나 ② 한국 사용자 영향 ③ 시장 영향
- CTA: 댓글 유도 + 채널 구독 자연스럽게
- 톤: 친근한 동네 형/누나, 존댓말. 전문 용어는 한 번 풀어 설명.
- 영어 약어는 한 번씩 풀어쓰기 (예: "GPT-5, 그러니까 챗지피티 5세대")
```

ChatGPT 출력 검수 → 문장 다듬기 → 발음 어색한 부분 띄어쓰기 조정.

### 단계 2. 얼굴 영상 — HeyGen Photo Avatar (영상당 5분)

**1회 셋업 (한 번만):**
- HeyGen Photo Avatar 등록 (본인 셀카 정면)
- Voice 등록 (본인 1분 보이스 클립 업로드)

**매번 작업:**
1. HeyGen 대시보드 → New Video → Avatar Video
2. Avatar 픽 + Voice 픽 + 스크립트 텍스트 붙여넣기
3. 9:16 비율 + 배경 (Plain white 또는 blurred desk)
4. Generate → 5~10분 대기 → 다운로드

**팁:**
- 한국어 발음 어색하면 "Pronunciation" 기능으로 단어 단위 수정
- 영어 약어 (GPT, Anthropic 등)는 한 번씩 한글 발음 표기 (예: "지피티-파이브")

### 단계 3. B-roll 생성 — Luma Ray 2 (영상당 15분)

뉴스 시각화 컷 4~6개 생성.

**시각화 카테고리별 프롬프트 (prompts.json 참조):**
- 모델 발표 → AI 칩 클로즈업, 데이터 흐름, 신경망 추상
- 가격/요금 변경 → 동전/숫자 모션, 그래프
- 회사 소식 → 빌딩, 로고 추상화, 회의실
- 사용자 영향 → 노트북 화면, 사람이 도구 쓰는 손 클로즈업

각 5초 × 4~6개 = 총 20~30초 B-roll.

### 단계 4. 편집 — CapCut (영상당 20~30분)

**컷 구조 표준:**
```
[0~5초]   훅 (B-roll 1 + 큰 자막 "GPT-5 출시 ✨")
[5~15초]  포인트 1 (B-roll 2 메인 + HeyGen 얼굴 우하단 작게 + 자막)
[15~30초] 포인트 2 (B-roll 3~4 + HeyGen + 키워드 모션 자막)
[30~45초] 포인트 3 (B-roll 5 + HeyGen + 비교 그래프)
[45~55초] CTA (HeyGen 얼굴 크게 + "여러분은 어떻게 생각하세요?")
[55~60초] 채널 시그니처 (워터마크 강조)
```

**CapCut 체크리스트:**
- [ ] 자동 자막 ON + 중요 단어 키컬러 강조 (오렌지 #FF6B35 추천)
- [ ] HeyGen 얼굴 영상은 우하단 25% 크기 (PIP)
- [ ] B-roll이 메인 화면 (75% 차지)
- [ ] 모션 그래픽: 핵심 숫자/이름 등장 시 1초 줌인 효과
- [ ] BGM: 미니멀 비트 (Future Bass / Lo-fi Tech) -22dB
- [ ] 화면 좌상단 "DAY.[일자]" 워터마크
- [ ] 마지막 자막: "Made with sora2-video-saas | 더 많은 AI 영상은 프로필"

### 단계 5. 게시 (영상당 5분)

- 캡션: brand-voice.md의 "호기심 유발 훅" + 뉴스 키워드 + CTA
- 해시태그: hashtag-library.json의 `concept_4_ai_news` 5개 + `core_brand_tags` 3개
- 게시 시각: 월요일 19:00 KST (또는 뉴스 발생 후 12시간 내 즉시)
- 본인 X (Twitter)에 "방금 영상 올렸어요" 크로스 포스트

## 12시간 사이클 타임라인

```
T+0h    뉴스 발생 (X 알림 또는 모닝 모니터링)
T+1h    소스 검증 (공식 발표 / 데모 영상)
T+2h    ChatGPT 스크립트 작성 + 검수
T+3h    HeyGen 얼굴 영상 생성 (대기 시간 포함)
T+4h    Luma B-roll 4컷 생성 (병렬)
T+6h    CapCut 편집
T+8h    검수 + 캡션
T+10h   게시
T+12h   첫 댓글 응대 + 통계 모니터링
```

**12시간 안에 가능한 이유:** B-roll 생성을 ChatGPT/HeyGen 대기 중 병렬로 돌리면 실 작업은 3~4시간.

## 시리즈화

- **시즌 1: "AI 뉴스 60초"** — 매주 월요일
- 회차 누적 → 100회 달성 시 "AI 뉴스 100선" 컴필레이션 롱폼 영상
- 인기 영상은 캐러셀 인스타 포스트로 재활용

| 주차 | 앵글 |
|------|------|
| W1 | 모델 출시 (GPT-X, Claude X, Gemini X) |

| W3 | 기업 인수/투자 소식 |

## 리스크 관리

- 루머는 반드시 "루머라고 명시" 자막 추가
- 경쟁사 비교 시 객관적 데이터만 (벤치마크 점수 등)
- 스크린샷 출처 자막 표기 (X, 공식 블로그 등)
- 잘못된 정보 게시 시 → 24시간 내 정정 영상 + 원본 영상 캡션에 정정 안내
