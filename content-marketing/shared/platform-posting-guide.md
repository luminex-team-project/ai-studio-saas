# 플랫폼별 게시 가이드

## 비교표

| 항목 | TikTok | YouTube Shorts | Instagram Reels |
|------|--------|----------------|-----------------|
| 비율 | 9:16 | 9:16 | 9:16 |
| 권장 길이 | 15~60초 (스위트스팟 21~34초) | 30~60초 | 15~30초 |
| 캡션 길이 | 100자 이내 | 첫 줄 60자 (검색 노출) | 125자 이내 |
| 해시태그 | 5~8개 | 3~5개 (#shorts 필수) | 8~10개 |
| 게시 골든타임 (KST) | 평일 19~22시 / 주말 11~13시 | 평일 18~21시 | 평일 20~22시 |
| 사운드 | 트렌드 사운드 직접 사용 (필수) | 자체 BGM 가능, 음원 정책 보수적 | Reels 사운드 라이브러리 |
| 자막 | 큰 글씨 + 화면 중앙/상단 | 자동 자막 ON, 추가 텍스트 강조 | 캡션 + 자체 자막 병행 |
| CTA 위치 | 마지막 2~3초 | 마지막 5초 | 캡션 첫 줄 |
| 핵심 알고리즘 | 첫 1초 시청완료율 | 평균 시청 시간 | 저장 + 공유 + 시청완료 |

## 영상 1초 훅 작성 룰

**TikTok:** 화면에 큰 텍스트 1줄 ("이거 진짜 AI?") + 강한 제스처/컷
**Shorts:** 질문형 ("이거 보고 충격 받음") + 인물 클로즈업
**Reels:** 시각적 임팩트 (밝은 색상, 빠른 컷) + "기다려봐" 텍스트

## 캡션 템플릿

### TikTok (100자)
```
[감탄/충격 문구] + [제품/주제 한 줄] + [CTA]
#태그1 #태그2 ...
```
예: `이게 AI라고? 🤯 sora2-video-saas로 만든 화장품 광고. 프로필 링크에서 직접 만들어보세요!`

### YouTube Shorts (첫 줄 60자)
```
[검색 키워드 포함 첫 줄]

(빈 줄)

[2~3줄 본문 + CTA]
#shorts #관련태그
```
예: `AI로 만든 K뷰티 광고 60초만에 만드는 방법`

### Reels (125자)
```
[1줄 훅] 👇
[1줄 가치]
[CTA + 프로필 링크 안내]
.
.
#태그모음
```

## CTA (Call-to-Action) 풀

| 의도 | 문구 |
|------|------|
| 호기심 유발 | "어떻게 만들었는지 댓글 달아드림" |
| 가입 유도 | "프로필 링크에서 무료 100 크레딧 받기" |
| 저장 유도 | "다음 광고 만들 때 참고하세요 (저장 추천)" |
| 공유 유도 | "마케터 친구 태그하면 100% 좋아함" |
| 댓글 유도 | "다음 영상 컨셉 댓글로 골라주세요" |

## 게시 후 30분 액션 (알고리즘 부스트)

1. 게시 즉시 → 본인 계정에 좋아요 + 저장
2. 5분 내 → 다른 SNS (Twitter/X, 카카오 채널)에 공유
3. 첫 댓글 직접 작성 (질문형: "여러분은 어떻게 생각해요?")
4. 30분 내 들어오는 댓글 100% 응대 (이모지 + 한 줄)
5. 게시 24시간 후 통계 확인 → 평균 시청 70% 미만이면 다음 영상 훅 강화

## 플랫폼별 특이사항

### TikTok
- **사운드 필수:** 자체 BGM만 쓰면 도달률 급감. 반드시 TikTok 라이브러리 또는 트렌드 사운드.
- **수익화:** Creator Fund 한국 미지원. 라이브 + 협찬으로 수익화.
- **외부 링크:** 팔로워 1,000 이상 시 프로필에 링크 노출 가능.

### YouTube Shorts
- **수익화:** 1,000 구독자 + 90일 1,000만 조회수 시 광고 수익 가능 (한국 가능).
- **롱폼 연계:** Shorts → 채널 구독 → 롱폼 튜토리얼 영상으로 유도하면 LTV 상승.
- **검색:** 첫 줄에 검색 키워드 (예: "AI 광고 만드는 법") 넣기.

### Instagram Reels
- **링크 인 바이오:** 1.5만 팔로워 미만은 링크 스티커 사용 가능 (스토리). Reels에는 링크 안 박힘 → 캡션에 "프로필 링크" 안내.
- **저장 가중치 큼:** "저장하세요" CTA가 효과적.
- **DM 유도:** "DM '광고'라고 보내주세요" → DM 기반 리드 수집 가능.

## 실패하는 영상 패턴 (피해야 할 것)

- 첫 1초가 로고/오프닝 → 즉시 스킵
- 자막 없음 → 무음 시청자 이탈
- 9:16 아닌 1:1 또는 4:5 → 강제 레터박스 → 도달률 ↓
- BGM 음량 -6dB 이상 → 보이스오버 묻힘
- 해시태그 0개 또는 30개 이상 → 알고리즘 의심
